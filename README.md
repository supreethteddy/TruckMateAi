## 🚚 Overview

TruckMate AI is India's smartest logistics network connecting customers with nearby trucks using real-time AI matching. The AI predicts optimal routes, fuel costs, tolls, and pricing instantly. Shippers upload cargo details; TruckMate recommends the best truck, verifies driver reliability, tracks live status, and automates payments, invoices, and contracts.

## ✨ Key Features

### 🤖 AI-Powered Features
- **Real-time AI Matching**: Smart cargo-to-truck compatibility matching
- **Route Optimization**: Fuel cost calculations and toll predictions
- **Dynamic Pricing**: Market-based pricing with demand/supply analysis
- **AI Demand Forecasting**: Route demand prediction dashboard
- **Earnings Optimization**: Suggestions for drivers based on market trends

### 🎤 Voice & Accessibility
- **Voice-enabled Booking**: Support for Hindi and regional languages
- **Multi-language Support**: Hindi, English, Tamil, Telugu, Bengali, Marathi
- **WhatsApp Integration**: Easy communication for non-tech drivers
- **Accessibility Features**: Voice commands and regional language support

### 📱 Core Features
- **Live Tracking**: Real-time shipment tracking with progress indicators
- **Smart Search**: Advanced filters and AI-powered truck recommendations
- **Booking Management**: Active, completed, and cancelled bookings
- **Driver Profiles**: Detailed driver information with ratings and contact
- **Route Planning**: Complete journey timeline with ETA predictions

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/supreethteddy/TruckMateAi.git
   cd TruckMateAi
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5174`
